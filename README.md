# FFmpeg VVC GUI видео конвертер

![Version](https://img.shields.io/badge/version-0.27-blue.svg)
![Python](https://img.shields.io/badge/python-3.8+-blue.svg)

Реализация графического интерфейса (GUI) над FFmpeg для удобного кодирования видео с использованием современного высокоэффективного кодека **H.266 (VVC)**.

Программа избавляет от необходимости вводить длинные консольные команды, предоставляя интуитивно понятный визуальный интерфейс для работы с видеофайлами.

## ✨ Ключевые возможности

* **Кодирование в H.266:** Поддержка новейшего стандарта сжатия видео (VVC).
* **Drag & Drop:** Поддержка простого перетаскивания файлов прямо в окно программы для ускорения рабочего процесса.
* **Минималистичность:** Легковесный интерфейс без лишних зависимостей.

## 🛠 Требования и установка

Для корректной работы программы вам понадобится Python 3 и сам FFmpeg.

1. **FFmpeg:** Убедитесь, что FFmpeg установлен в вашей системе и добавлен в переменные среды (`PATH`), либо исполняемый файл `ffmpeg.exe` лежит в одной папке со скриптом.
2. **Зависимости Python:** Для работы функции Drag & Drop требуется установить дополнительную библиотеку. Откройте терминал и выполните:
